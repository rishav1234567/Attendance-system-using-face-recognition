# Face Recognition Attendance System

## Overview

This project is a Python-based Face Recognition Attendance System that uses OpenCV and the face_recognition library to identify registered users through a webcam and automatically mark their attendance.

## Features

* Real-time face detection
* Face recognition using stored images
* Automatic attendance marking
* CSV-based attendance records
* Easy to add new users

## Technologies Used

* Python
* OpenCV
* face_recognition
* NumPy

## Installation

1. Clone the repository:

```
git clone https://github.com/rishav1234567/Attendance-system-using-face-recognition.git
```

2. Create a virtual environment:

```
python -m venv .venv
```

3. Activate the virtual environment.

Windows:

```
.venv\Scripts\activate
```

4. Install dependencies:

```
pip install -r requirements.txt
```

5. Run the project:

```
python as.py
```
